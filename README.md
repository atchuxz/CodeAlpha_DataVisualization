
# 🍽️ Zomato Restaurant Data Visualization using Python

## 📌 Project Overview
This project focuses on analyzing and visualizing Zomato restaurant data using Python. The aim of this project is to identify restaurant trends, customer preferences, ratings distribution, online ordering trends, and popular restaurant locations through graphical visualizations.

This project was completed as part of the CodeAlpha Data Analytics Internship.

## 🚀 Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## 📂 Dataset Used
Dataset: Zomato Bangalore Restaurants Dataset
The dataset contains information about:
- Restaurant names
- Ratings
- Locations
- Online order availability
- Approximate cost
- Cuisines
- Table booking information

## 📊 Visualizations Included

### 1. Restaurant Ratings Distribution
Analyzed how restaurant ratings are distributed.

### 2. Online Order Availability
Visualized percentage of restaurants supporting online orders.

### 3. Top Restaurant Locations
Identified areas with the highest number of restaurants.

### 4. Cost vs Rating Analysis
Studied the relationship between restaurant pricing and ratings.

### 5. Correlation Heatmap
Displayed relationships between numerical features.

### 6. Top Cuisines
Visualized the most popular cuisines among restaurants.

## 📈 Key Insights
- Most restaurants support online ordering.
- Restaurant ratings are mostly between 3.5 and 4.5.
- Some locations contain significantly more restaurants than others.
- Mid-range restaurants tend to receive better ratings.
- Popular cuisines dominate the restaurant market.
